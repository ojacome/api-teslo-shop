# TesloShop Api

Clonar el proyecto

Instalar dependencias
```
npm i
```

Para levantar postgres en docker
```
docker-compose up -d
```

Clonar el archivo __.env.template__ y renombrarlo __.env__ y personalizar variables

Levantar el proyecto
```
npm run start:dev
```
